How It Works

This is an open source personal project making use of www.postalpincode.in in a try to learn and implement Jquery and Bootstrap.

The search can be done using Pincode and Place name.
The search text is used to do an api call for both Pincode and Place name.

The results received are concatinated and displayed on UI.

In case of error/No result an information is shown on UI. Hope this helps. Contact us in case of any clarifications Contact Us.
